Problems

Team: Andrés Mejía, Ana Echavarría and Sebastián Arcila.

Conga Line
----------
Difficulty: Easy
Intended solution: Greedy algorithm
Sample solution complexity: O(n) per test case

Stones
------
Difficulty: Medium
Intended solution: Dynamic programming or Game theory
Sample solution complexity: O(n^2) per test case

Secret word
-----------
Difficulty: Hard
Intended solution: Modification of Knuth-Morris-Pratt algorithm
Sample solution complexity: O(n) per test case
